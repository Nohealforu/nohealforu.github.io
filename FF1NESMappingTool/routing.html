FF1 NES Route Simulation Tool 0.3.6


Encounter format: "Encounter encounter_number (danger_level) (hp_allowed) (action)"
Source code on the Github repository.
This free tool is for learning about the game systems and planning routes for Speedrunning, it is not a recreation of or substitute for the game.
If you're looking to play the original game, I highly recommend purchasing the Pixel Remaster version.
Thank you to fcoughlin for creating a routing tool I used for reference of much of the game logic.
